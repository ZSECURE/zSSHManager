# zSSHManager

## How to run

``` bash
python3 zSSHManager.py
```
## Running

```bash
Menu:

1. List Hosts
2. Run Command
3. Open Shell
4. Add Server
5. Remove Server
6. Exit

Enter your choice: 
```
## Example servers.txt file
The format is IP_Address,Username,Password
``` bash
192.168.0.1,root,toor
192.68.1.1,root,toor
192.168.10.1,root,toor
```
